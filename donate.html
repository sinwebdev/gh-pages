Donate

Please scan QR code below or click Donate to make a donation.

All donations to Woodcroft Wildspace very much appreciated.

SumUp does not pass on any personal details to us about your donation, only the code that it provides to you on completing the transaction.

It would be helpful for us to be able to relate donations to donors, for example when making a booking, so if you could please email the code to enquiries@woodcroft.org.uk with a short note about your kind donation that would be helpful.

If you would like to remain anonymous that is of course fine, thank you for your donation.

Either way, many thanks for your donation. We will put it to good use in maintaining the Wildspace.
